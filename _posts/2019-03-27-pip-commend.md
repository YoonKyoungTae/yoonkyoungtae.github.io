# 명령어
pip에 대해 잘 모르기 때문에 다시 사용할 법한 명령어를 정리한다.  
이 페이지는 계속 수정할 예정이다.

#### 패키지 설치

    pip install 패키지명


#### 패키지 삭제

    pip uninstall 패키지명


#### 패키지 업데이트

    pip install --upgrade 패키지명


#### 설치된 패키지 목록확인

    pip list
